# Phase 4: Binary Replacement

## Goal

Extract the binary from the downloaded archive and atomically replace the running executable, with platform-specific handling for Windows.

## Tasks

### 4.1 Unix strategy (`replace.go`)

**File:** `internal/update/replace.go` (build tag: `//go:build !windows`)

```go
func ReplaceBinary(archivePath, binaryPath string) error
```

Implementation:
1. Detect archive format by extension (`.tar.gz`).
2. Open with `gzip.NewReader` → `tar.NewReader`.
3. Iterate entries, find the one named `scry` (no directory prefix — GoReleaser puts it at root).
4. Write to temp file in same directory: `filepath.Dir(binaryPath)` + random suffix.
5. `os.Chmod(tempPath, 0755)`.
6. `os.Rename(tempPath, binaryPath)` — atomic on same filesystem.
7. On any failure after temp file creation: `os.Remove(tempPath)`, return `ErrReplaceFailed`.

### 4.2 Windows strategy (`replace_windows.go`)

**File:** `internal/update/replace_windows.go` (build tag: `//go:build windows`)

```go
func ReplaceBinary(archivePath, binaryPath string) error
```

Implementation:
1. Detect archive format by extension (`.zip`).
2. Open with `zip.OpenReader`.
3. Find entry named `scry.exe`.
4. Write to temp file in same directory.
5. Rename current binary: `os.Rename(binaryPath, binaryPath+".old")`.
6. Rename new binary: `os.Rename(tempPath, binaryPath)`.
7. Best-effort delete of `.old` file.
8. If step 5 or 6 fails: return `ErrReplaceFailed` wrapping "try running with elevated privileges or replace manually".

### 4.3 Archive format detection

Shared helper (can live in `replace.go` with the Unix build tag, or a separate `archive.go` without build tags if needed by both):

```go
func isTarGz(path string) bool { return strings.HasSuffix(path, ".tar.gz") }
func isZip(path string) bool   { return strings.HasSuffix(path, ".zip") }
```

In practice, Unix always gets tar.gz and Windows always gets zip (per GoReleaser config), so each platform file only needs to handle its own format.

### 4.4 Write tests

**File:** `internal/update/replace_test.go`

- Create a real tar.gz containing a fake binary, call `ReplaceBinary`, verify content and permissions.
- Create a real zip containing a fake binary, call `ReplaceBinary`, verify content.
- Test with read-only target directory → expect `ErrReplaceFailed`.
- Test that temp files are cleaned up on failure.

Helper to create test archives:

```go
func createTestTarGz(t *testing.T, dir, name, content string) string
func createTestZip(t *testing.T, dir, name, content string) string
```

## Acceptance Criteria

- tar.gz extraction produces correct binary with 0755 permissions
- zip extraction produces correct binary
- Atomic rename replaces the target file
- Failures clean up temp files
- Read-only directory returns `ErrReplaceFailed`
