# Phase 5: Update Orchestrator

## Goal

Wire all components together into a single `Run()` function that the CLI calls.

## Tasks

### 5.1 Create `Run()` function

**File:** `internal/update/update.go`

```go
func Run(ctx context.Context, currentVersion string, checker ReleaseChecker) (*Result, error)
```

### 5.2 Implement `isNewer()` semver comparison

```go
func isNewer(remote, current string) bool
```

- Strip `v` prefix from both.
- Split on `.` to get `[major, minor, patch]`.
- Parse each as int, compare left-to-right.
- Return true only if remote is strictly greater.
- If parsing fails (non-numeric), fall back to string comparison.

### 5.3 Implement `deriveAssetName()`

```go
func deriveAssetName(tag string) string
```

- Strip `v` prefix from tag to get version number.
- Pattern: `scry_<version>_<runtime.GOOS>_<runtime.GOARCH>.<ext>`
- Extension: `.zip` if `runtime.GOOS == "windows"`, `.tar.gz` otherwise.
- Example: tag `v1.2.3` on linux/amd64 â†’ `scry_1.2.3_linux_amd64.tar.gz`

### 5.4 Implement write permission check

```go
func writable(dir string) bool
```

- Attempt to create and immediately remove a temp file in `dir`.
- If creation fails â†’ not writable.
- This is more reliable than checking file mode bits (handles ACLs, SELinux, etc.).

### 5.5 Wire the pipeline

Full flow in `Run()`:
1. Check `currentVersion != ""` â†’ else `ErrDevBuild`
2. `checker.LatestRelease(ctx)` â†’ get release
3. `isNewer(release.Tag, currentVersion)` â†’ if false, return `Result{UpToDate: true}`
4. `deriveAssetName(release.Tag)` â†’ find in `release.Assets`
5. Find `checksums.txt` in `release.Assets`
6. If either missing â†’ `ErrAssetNotFound`
7. Resolve binary path via `os.Executable()`
8. Check `writable(filepath.Dir(binaryPath))` â†’ else `ErrPermission`
9. Download archive and checksums to binary's directory
10. `VerifyChecksum(...)` â†’ on error, cleanup and return
11. `ReplaceBinary(...)` â†’ on error, cleanup and return
12. Cleanup temp files, return `Result{OldVersion, NewVersion}`
