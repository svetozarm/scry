# Phase 1: Foundation (Version Embedding + Errors)

## Goal

Establish the version injection mechanism and define the error types that the rest of the update module will use.

## Tasks

### 1.1 Add version variable to `main.go`

**File:** `main.go`

Add a package-level variable that `-ldflags` will populate:

```go
var version string
```

Pass it to the `cmd` package. The `cmd` package needs a `SetVersion(v string)` function or an exported variable so `update.go` can access it later.

**Approach:** Add `cmd.Version = version` before `cmd.Execute()`. Add `var Version string` to `cmd/root.go`.

### 1.2 Update Makefile

**File:** `Makefile`

Change the `build` target to inject version from git tags:

```makefile
VERSION ?= $(shell git describe --tags --always --dirty 2>/dev/null || echo "dev")

build:
	CGO_ENABLED=0 go build -ldflags="-s -w -X main.version=$(VERSION)" -o $(BINARY) .
```

### 1.3 Update `.goreleaser.yaml`

**File:** `.goreleaser.yaml`

Add version ldflags to the builds section:

```yaml
builds:
  - ldflags:
      - -s -w -X main.version={{.Version}}
```

GoReleaser's `{{.Version}}` template resolves to the tag without the `v` prefix. Use `{{.Tag}}` instead to preserve the `v` prefix for consistency with the API response.

### 1.4 Create error definitions

**File:** `internal/update/errors.go`

```go
package update

import "errors"

var (
    ErrUpdateAPI        = errors.New("update: GitHub API error")
    ErrChecksumMismatch = errors.New("update: checksum mismatch")
    ErrAssetNotFound    = errors.New("update: platform asset not found in release")
    ErrPermission       = errors.New("update: binary path not writable")
    ErrReplaceFailed    = errors.New("update: failed to replace binary")
    ErrDevBuild         = errors.New("update: no version embedded (development build)")
)
```

### 1.5 Write error tests

**File:** `internal/update/errors_test.go`

Verify that `fmt.Errorf("context: %w", ErrUpdateAPI)` is matchable via `errors.Is()`. Simple but ensures the wrapping pattern works.

## Acceptance Criteria

- `go build -ldflags="-X main.version=v1.0.0"` produces a binary where `cmd.Version == "v1.0.0"`
- `make build` injects the git tag as version
- GoReleaser config injects the tag
- All 6 sentinel errors are defined and testable via `errors.Is()`
